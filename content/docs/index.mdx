Amarsia is a platform to build, deploy, and run AI assistants. You design your assistant in the dashboard — prompt, tools, knowledge — deploy it, and get an API to run it from any app.

## Dashboard — start building

Start building your assistant or automation on the Amarsia dashboard.

- [Create assistant](https://amarsia.com/dash/assistants/new) — build a new AI assistant, set system instructions, attach knowledge, and connect actions
- [Create knowledge base](https://amarsia.com/dash/knowledge-base) — upload documents or URLs your assistant can reference when answering
- [Create action](https://amarsia.com/dash/actions/new) — connect external APIs or MCP servers your assistant can call as tools
- [Create API key](https://amarsia.com/dash/api_keys) — generate credentials to authenticate requests from your app
